# PWA Icons Placeholder

Bu klasör Progressive Web App (PWA) ikonları için ayrılmıştır.

## Gerekli Boyutlar:

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Hızlı Oluşturma:

1. 512x512 master icon oluştur
2. https://www.pwabuilder.com/imageGenerator kullan
3. Tüm boyutları buraya kopyala

## Design Guidelines:

- Kare format (1:1)
- Maskable area: %80 safe zone
- Transparan veya solid background
- TravelMatch brand colors: #00FF88 (primary), #050505 (background)
