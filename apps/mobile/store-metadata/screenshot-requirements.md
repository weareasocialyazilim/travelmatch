## Play Store Graphics

### Required Assets

| Asset Type | Dimensions | Format |
|------------|------------|--------|
| Feature Graphic | 1024 × 500 | PNG/JPEG |
| Hi-res Icon | 512 × 512 | PNG (32-bit) |
| Phone Screenshots | 320-3840 px | PNG/JPEG |
| 7" Tablet | 1024 × 500+ | PNG/JPEG |
| 10" Tablet | 1024 × 500+ | PNG/JPEG |

### Feature Graphic Guidelines
- Showcase app's main value proposition
- Include app logo and tagline
- Use brand colors (#6366F1 primary)
- Travel/experience imagery
- Text should be legible on small displays

---

## Screenshot Design Guidelines

### Brand Colors
- Primary: #6366F1 (Indigo)
- Secondary: #10B981 (Emerald)
- Background: #FFFFFF
- Text: #1F2937

### Typography
- Headlines: SF Pro Display Bold (iOS) / Roboto Bold (Android)
- Body: SF Pro Text (iOS) / Roboto (Android)

### Content Requirements
- No placeholder text (lorem ipsum)
- Use realistic Turkish names and content
- Show actual app UI (no mockups)
- Include status bar with realistic time, battery, signal
- Device frames optional but recommended

### Localization
- Turkish (tr-TR): Primary
- English (en-US): Secondary
